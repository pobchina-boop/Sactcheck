SACTCheck v0.16.2 protocol-preview reliability fix

1. Copy everything in this folder into the root of the existing SACTCheck repository.
2. Choose Replace files in destination.
3. Stop and restart Live Server.
4. Press Ctrl+F5 in the browser.
5. Click Preview protocol JSON.

The preview screen is now present directly in index.html and the JavaScript modules are cache-busted. If a module is missing, the screen shows a visible diagnostic instead of silently doing nothing.
