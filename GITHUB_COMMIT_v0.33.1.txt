Description
Improves assessment transparency by displaying the entered patient result directly beside the exact encoded protocol cutoff for every triggered generic JSON rule.

The update preserves the rapid partial-assessment workflow, blood-first input order, compact optional fields, TCHP single-value behaviour, emetogenic-risk framework, official NCCP PDF checks and the existing 58-protocol catalogue.

Adds dedicated regression testing for condition preservation and threshold rendering. The full cumulative automated test suite passes.
