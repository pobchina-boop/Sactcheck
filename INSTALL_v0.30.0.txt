Installation
1. Confirm the current repository is committed and pushed.
2. Extract SACTCheck_v0.30.0_drop_in_update.zip.
3. Open SACTCheck_v0.30.0_drop_in.
4. Copy everything inside that folder into the existing Sactcheck repository.
5. Choose Replace the files in the destination.
6. Restart VS Code Live Server.
7. Press Ctrl + F5 in the browser.
