SACTCheck v0.14 drop-in update
================================

1. Keep your current Sactcheck repository folder.
2. Extract the supplied drop-in ZIP.
3. Copy its contents into the root of your existing Sactcheck folder.
4. When Windows asks, choose "Replace the files in the destination".
5. Open the repository in VS Code and run it with Live Server, or open the GitHub Pages version.
6. In the regimen library, look for:
   - Pembrolizumab + Carboplatin/Paclitaxel → AC
   - Olaparib monotherapy
   Their cards should now show "Generic assessment available".
7. Test with the "Load demonstration values" button before entering manual values.
8. In GitHub Desktop, review the changed files, then commit and push when satisfied.

Optional automated test
-----------------------
Open a terminal in the repository and run:

node tests/engine.test.js

Expected result: 13 engine tests passed.

Important
---------
Run the app through GitHub Pages, VS Code Live Server or another HTTP server.
Opening index.html directly from the Windows file system can prevent JSON files from loading.

This remains a demonstration prototype pending consultant, oncology-pharmacy and formal software validation.
