SACTCheck v0.17 installation
=============================

1. Confirm the existing SACTCheck changes are committed and pushed on the develop branch.
2. Extract SACTCheck_v0.17_drop_in_update.zip.
3. Open the extracted SACTCheck_v0.17_drop_in folder.
4. Copy everything inside that folder into the root of the existing Sactcheck repository.
5. Choose "Replace the files in the destination" when Windows asks.
6. Open index.html with VS Code Live Server.
7. Press Ctrl+F5 once to clear older cached JavaScript.
8. Complete the manual checks in UPDATE_INSTRUCTIONS.txt before committing.
