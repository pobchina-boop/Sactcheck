SACTCheck v0.27.0 — Drop-in installation

1. Commit or back up the current repository.
2. Extract the v0.27 drop-in ZIP.
3. Copy everything inside the extracted folder into the existing Sactcheck repository.
4. Choose Replace files in destination.
5. Restart Live Server and refresh the browser with Ctrl+F5.
6. Check durvalumab and atezolizumab: low ANC/platelets must not generate automatic protocol holds.
7. Check carboplatin + etoposide: indication and etoposide schedule must appear in Protocol and treatment context.
8. Run npm test before committing.
