SACTCheck v0.28.0 — Batch 4 protocol expansion

BASE VERSION
This drop-in update is cumulative from v0.27.2. Apply it to the repository currently running v0.27.2.
The full-project ZIP is the safer cumulative backup if the local repository version is uncertain.

INSTALLATION
1. Ensure the current repository changes have been committed and pushed.
2. Extract SACTCheck_v0.28.0_drop_in_update.zip.
3. Open the extracted SACTCheck_v0.28.0_drop_in folder.
4. Copy everything INSIDE that folder into the existing Sactcheck repository root.
5. Choose Replace the files in the destination.
6. Restart VS Code Live Server.
7. In the browser press Ctrl+F5.

CHECKS
- The top badge should show Version 0.28.0 · Batch 4 expansion.
- The JSON catalogue should contain 28 protocols.
- Search for NCCP 00722, 00726, 00831, 00897 and 00558.
- Each new card should open a generated assessment and an official NCCP PDF.
- NCCP 00897 should hide ANC, platelets, cisplatin renal and hearing inputs when the durvalumab-maintenance phase is selected.
- Pembrolizumab 400 mg monotherapy should not show numerical ANC or platelet hold thresholds.
