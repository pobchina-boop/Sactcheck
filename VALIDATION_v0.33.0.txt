SACTCheck v0.33.0 validation

Scope
- Rapid assessment interface changes applied to the generic JSON assessment screen.
- Emetogenic-risk traffic-light framework added without inferring clinical classifications.

Verified
- Blood thresholds render before other clinical and treatment-context inputs.
- No HTML required attributes remain on generated protocol inputs.
- Blank context and clinical fields do not block independent single-value assessment.
- Optional fields use compact clickable rows with a visible Not assessed state.
- All active catalogue protocols have a central emetogenic mapping entry.
- All emetogenic entries are deliberately pending until the antiemetic proforma is supplied.
- Existing protocol, duplicate, official-PDF and TCHP regression tests remain in the cumulative suite.

Clinical governance
- Traffic-light colours are not yet clinically populated. Grey means unclassified/pending proforma mapping.
- No pending value should be interpreted as low emetogenic potential.
- Encoded protocols remain pending formal consultant and oncology-pharmacy validation.
