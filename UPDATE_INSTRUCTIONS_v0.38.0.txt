SACTCheck v0.38.0 update instructions

1. Extract SACTCheck_v0.38.0_Complete_Prostate_Library.zip.
2. Replace the existing hosted Sactcheck folder with the extracted Sactcheck folder.
3. Commit and push the complete folder, including the new protocols/genitourinary directory and all updated JavaScript/data/test files.
4. Confirm the app badge reads “Version 0.38.0 · Complete prostate library”.
5. Select Genitourinary / Men’s cancers and confirm the prostate catalogue is populated.
6. Search for generic and trade names, for example abiraterone/Zytiga, enzalutamide/Xtandi, goserelin/Zoladex and radium-223/Xofigo.
7. Open a prostate protocol and confirm the Official NCCP PDF action is present.
8. Enter a single relevant value only and confirm a partial assessment is generated without completing unrelated fields.
9. Check a protocol with tiered renal guidance and confirm it uses selectable renal categories rather than exact numerical entry.
10. Clear the browser cache or reload the installed Home Screen app if an older version remains visible.
