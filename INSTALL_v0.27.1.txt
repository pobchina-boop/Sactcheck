SACTCheck v0.27.1 — Duplicate-card hotfix

1. Extract SACTCheck_v0.27.1_drop_in_update.zip.
2. Open the extracted SACTCheck_v0.27.1_drop_in folder.
3. Copy everything inside it into the existing Sactcheck repository.
4. Choose Replace the files in the destination.
5. Restart Live Server and press Ctrl+F5.
6. Confirm the regimen library contains one Carboplatin + Etoposide card showing NCCP 00271 Version 7.

The public card now launches the JSON assessment. The older legacy assessment code remains in the repository as a fallback but no longer creates a second library card.
