SACTCheck v0.37.0 installation
===============================

1. Commit or back up the existing SACTCheck repository before replacing files.
2. Extract SACTCheck_v0.37.0_Platform_Standardisation.zip.
3. Open the extracted Sactcheck folder.
4. Copy its contents into the root of the existing Sactcheck repository and replace existing files when prompted.
5. Run: npm run protocols:build
6. Run: npm test
7. Open index.html using VS Code Live Server or the existing GitHub Pages deployment.
8. Hard-refresh once (Ctrl+F5 on Windows) to clear older JavaScript cache entries.
9. Confirm the catalogue badge reads "Version 0.37.0 · Platform standardisation".
10. Review RELEASE_NOTES_v0.37.0.md and STANDARDISATION_AUDIT_v0.37.0.md before any pilot use.

Clinical boundary
-----------------
The release remains a decision-support prototype. Automated tests do not establish clinical authorisation. Verify every output against the current official NCCP protocol and local oncology-pharmacy policy.
