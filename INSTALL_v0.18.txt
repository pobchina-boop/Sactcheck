SACTCheck v0.18 installation

1. Replace the existing project files with the contents of this package.
2. Restart Live Server.
3. Refresh the browser with Ctrl+F5.
4. Open Weekly paclitaxel from the normal regimen catalogue. It should now open the JSON protocol assessment directly.
5. Olaparib continues to open through its JSON catalogue card.

The legacy weekly paclitaxel assessment remains in the source temporarily for rollback/reference, but is no longer the catalogue launch pathway.
