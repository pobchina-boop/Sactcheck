Summary:
Version 0.33.0 · Rapid assessment interface and emetogenic-risk framework

Description:
Reworks the SACTCheck protocol assessment interface around rapid day-ward use. Blood thresholds are now the first and most prominent clinical inputs, with ANC, platelets, haemoglobin and WBC-type fields prioritised automatically when encoded by a regimen.

Removes browser-level mandatory-entry restrictions so any available single value can be assessed without entering cycle number or unrelated treatment context. Blank fields remain explicitly unassessed and are never assumed normal. Other clinical and context inputs are presented as compact clickable rows showing Not assessed until opened and completed.

Adds a central regimen-level emetogenic traffic-light framework to catalogue cards and protocol pages. All 58 active regimens are currently grey and marked as awaiting proforma mapping; no high, moderate or low category has been inferred before the supplied antiemetic proforma is incorporated.

Retains the existing 58-protocol catalogue, duplicate safeguards, official NCCP PDF validation, TCHP single-value regression correction and cumulative automated test suite.
