SACTCheck v0.22 full-project release

This release promotes NCCP 00321 Version 9 XELOX / CAPOX to the live generic JSON assessment pathway.

Recommended workflow:
1. Back up the current repository folder.
2. Replace the working project files with this full-project release.
3. Run npm test.
4. Open index.html locally and confirm that XELOX / CAPOX opens the JSON assessment and the Official NCCP PDF link.
5. Commit and push the tested release to GitHub.

Clinical and oncology-pharmacy validation remain pending. Always confirm the current official NCCP PDF and apply independent clinical judgement.
