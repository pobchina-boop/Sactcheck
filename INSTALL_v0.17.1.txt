SACTCheck v0.17.1 installation

1. Extract the full-project ZIP.
2. Open the Sactcheck folder in VS Code.
3. Start Live Server from index.html.
4. Hard-refresh the browser (Ctrl+F5).
5. Open the JSON shadow assessment for olaparib or weekly paclitaxel.
6. Enter a single clinical value and run the assessment.

Expected behaviour
- No browser “Please fill out this field” block.
- A restrictive single value can generate an immediate hold/reduce/withhold result.
- Missing unrelated fields appear as “Not assessed” coverage gaps.
- A single normal value cannot generate overall regimen clearance.

Validation
Run: npm test
